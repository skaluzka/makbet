**01.dummy**
------------

The main goal of this example is to... do nothing special.  The file
``01.dummy/Makefile`` (called also a **makbet's** scenario file) contains list
of all tasks which are specific for this example.  All defined tasks are just
a fake tasks (please check the content of
``examples/01.dummy/tasks/generic-task.sh`` script).  This example can be run
at any time with minimal requirements and effort.
